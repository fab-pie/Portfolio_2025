## À propos

Ce portfolio présente mes projets, expériences et parcours en tant qu'étudiant ingénieur.

## Développement local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.

## Technologies

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Three.js
- Framer Motion

## Déploiement

Le projet est automatiquement déployé sur Vercel à chaque push sur `main`.
